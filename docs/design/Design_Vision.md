# PHANTOM Design Vision

## Образ продукта
PHANTOM должен ощущаться не как “хакерская утилита”, а как:
- премиальный,
- спокойный,
- технологичный,
- доверительный,
- очень чистый,
- modern privacy product.

Не дешёвый cyberpunk, не кислотный “дарк веб”, не Telegram clone.

## Ключевой вайб

### Ассоциации
- холодная уверенность;
- ночной город;
- matte glass;
- deep indigo / obsidian / graphite;
- светящийся акцент, но не неоновый цирк.

## Визуальные принципы

### 1. Dark-first
Основная тема тёмная.
Не мрачная, а глубокая и дорогая.

### 2. Воздух
Много отступов, крупная типографика, немного элементов на экране.

### 3. Status visibility
Сеть, transport, secure state и delivery state должны быть видны, но не шуметь.

### 4. Privacy feel
Интерфейс должен внушать ощущение:
“здесь всё под контролем, тихо и безопасно”.

### 5. Не копировать Telegram буквально
Можно брать удобство паттернов, но не повторять один в один:
- иерархию экранов,
- цветовые решения,
- мелкие детали поведения.

## Цветовое направление

### Base
- background: near-black / graphite
- surfaces: dark slate / muted navy
- borders: ultra-low contrast

### Accent
- deep indigo
- electric violet (очень дозированно)
- muted cyan как secondary state

### Semantic
- success: мягкий emerald
- warning: warm amber
- danger: restrained red

## Типографика

Нужен clean modern grotesk/sans feel.
Приоритет:
- читаемость,
- спокойствие,
- premium look.

## Motion

Анимации:
- короткие,
- мягкие,
- physics-light,
- без перегруза.

Ощущение: secure system, not toy app.

## Первые экраны, которые должны “зацепить”

1. Onboarding
2. Username creation
3. Empty chat list
4. Filled chat list
5. Main chat screen
6. Contact safety screen
7. Nearby / offline screen
8. Settings with transport state
